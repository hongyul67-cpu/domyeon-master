# 📐 도면읽기 마스터 아카데미

자동설비 기능사 대비 **도면 읽기 학습 게임** 모음입니다. 하나의 HTML 파일로 동작하며, 설치·인터넷 없이 브라우저(휴대폰 포함)에서 바로 실행됩니다.

## 🎮 플레이하기

- **온라인**: https://hongyul67-cpu.github.io/domyeon-master/
- **오프라인**: `index.html` 파일을 브라우저로 열기

## 구성

세 훈련장을 돌며 자격증을 모으고, 마지막 종합 자격시험까지 통과하면 **마스터 기술자**가 됩니다. 진행 상황은 브라우저에 자동 저장됩니다.

| 훈련장 | 내용 |
|--------|------|
| 📏 선의 종류 마스터 | 외형선·숨은선·중심선·치수선·절단선·가상선 구분 (4개 모드) |
| 📐 우주선 조립 도크 | 치수 기호 해독 · 치수선 구분 · 치수 계산 · 오류 찾기 (4개 도크) |
| 🔧 로봇 정비소 | 나사·볼트·기어·베어링 등 기계요소 약도 익히기 (4개 미션) |
| 🎯 종합 자격시험 | 세 도메인 문제를 무작위 혼합 출제 (자격증 3개 획득 시 해금) |

## 특징

- 계급 시스템(견습생 → 마스터 기술자)과 자격증 보관함
- 휴대폰 터치 최적화 — 넓은 터치 영역, 선택 후 **제출** 방식(실수 클릭 방지)
- 진행 상황 자동 저장(localStorage)

## 파일

- `index.html` — 통합 실행본 (위 3개 게임 + 종합시험)
- `선의_종류_마스터.html`, `치수_읽기_마스터.html`, `기계요소_약도_마스터.html` — 개별 원본 게임
